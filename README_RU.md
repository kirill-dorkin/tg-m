# Telegram Cadence — инструмент для маркетинга в Telegram

Разработчики: NotVaishav, AtmegaBuzz, Gagandheep  
Софт предоставлен компанией Vscale Consulting LLP.

В отдельной ветке `2fa` доступна возможность добавления аккаунтов с двухфакторной аутентификацией.

## Возможности
- Подключение нескольких аккаунтов Telegram с поддержкой двухфакторной аутентификации.
- Создание неограниченного числа "каденций" — последовательностей сообщений.
- Добавление сколь угодно большого числа сообщений внутри каденции.
- Импорт и сопоставление сообщений из Google-таблиц.
- Запуск каденции сразу в нескольких группах Telegram.
- Планирование времени отправки сообщений и всей каденции целиком.
- Назначение отдельных аккаунтов для отправки разных сообщений.
